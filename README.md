# Titanic Survival Data Analysis (EDA)
A quick Exploratory Data Analysis (EDA) project built using Python and Google Colab.

## Objective
To identify the key factors that influenced survival rates during the Titanic disaster using data visualization and statistical summaries.

## Tools Used
- **Environment:** Google Colab (Cloud-based Jupyter Notebooks)
- **Language:** Python 3
- **Libraries:** Pandas (Data Manipulation), Seaborn & Matplotlib (Visualization)

##  Key Insights
- **The "Women and Children First" Rule:** Females in 1st class had a survival rate of over 90%, compared to less than 20% for males in 3rd class.

##  How to View
Click the `.ipynb` file above to view the analysis and visualizations directly in your browser.
